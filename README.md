# Smart-Study-Recommendation-System

## Overview
This project is an AI-based system that helps students identify their weak subjects and improve their study strategy.

## Features
- Takes marks and study hours as input
- Predicts weak subject using Machine Learning
- Provides study recommendations
- Includes both GUI and command-line versions

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Tkinter

## How to Run: 
### Simple Version (Recommended)
pip install pandas scikit-learn  
python simple_version.py  

### GUI Version (Run Locally)
python project.py  

## Output
- Weak subject prediction  
- Study recommendation  

## AI Concepts Used
- Supervised Learning  
- Decision Tree Classifier  
- Intelligent Agents  

## Note
GUI version may not run in browser environments.
